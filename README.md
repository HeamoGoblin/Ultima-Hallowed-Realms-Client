# Ultima-Hallowed-Realms-Client
Game Client For Ultima Hallowed Realms
---------------------------------------------------------------------------------------------
How to play/Use this git

The Git is meant to be used together with a server package you can download in seperate Git release area Here: https://github.com/HeamoGoblin/Ultima-Hallowed-Realms

1.)To play: clone this git (or any other branch of this git you want) on your local hard drive

2.)make sure your cloned git is in c:\Ultima-Hallowed-Realms

3.)Download the drop-in additional files in the release tab, extract everything inside the Ultima-Hallowed-Realms-Client folder in the archive to your C:\Ultima-Hallowed-Realms folder

4.)Run the runme options to launch the client.
