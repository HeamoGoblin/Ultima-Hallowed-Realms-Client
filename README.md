# Ultima-Hallowed-Realms-Client
Game Client For Ultima Hallowed Realms
---------------------------------------------------------------------------------------------
How to play/Use this git

This git never needs to be cloned we will only use it to push text files that will hold the client side updates for offline players all online player will get the updates automatically with the shards IPL = Installer, Patcher, Launcher!

The Git is meant to be used together with a server package you can download in seperate Git release area Here: https://github.com/HeamoGoblin/Ultima-Hallowed-Realms

1.)To play: Download the client from Google Drive Here: (Coming Soon)

2.)Make sure you unzip the client inside the Server Git that you had cloned it should be located at C:\Ultima-Hallowed-Realms

3.)Now you will have two folders one being the Server the other being your Client

4.)Now open your Client Folder inside of their you will find another folder called TazUO click on the TazUO Luancher.

5.)Now you are ready to play using the testing option from the drop down menu is for offline play using the hallowed realms option from the drop down will allow you to play on the online shard with all online players!
